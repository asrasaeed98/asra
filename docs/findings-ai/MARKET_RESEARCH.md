# Market research — positioning

## Closest competitors

| Product | Strength | Gap vs Findings.ai |
|---------|----------|-------------------|
| [Data Commons](https://datacommons.org) | NL + unified public stats | Pre-joined graph, not raw 2-dataset join + sklearn |
| [Mobus](https://github.com/mobus-ai/Mobus) | 20+ source search + license | MCP tool, not consumer wizard |
| [Dataset Explorer](https://github.com/peter-products/dataset-explorer) | 200K schema semantic search | Discovery only |
| [Julius AI](https://julius.ai) | Upload + chat + charts | BYO data; weaker license/audit story |
| [Auto-Analyst](https://www.autoanalyst.ai) | Stats + sklearn agents | Not open-data catalog first |
| data.gov | Catalog | No auto analysis |

## Differentiators

1. Strict CC0/PD catalog gate
2. Deterministic metrics + labeled AI summary
3. Guided 1–2 dataset join + sampling UX
4. Clustering/anomalies on public data in one flow

## Positioning statement

**Findings.ai** is the easiest way to go from **trusted open datasets** to **verified insights and visuals** — without spreadsheets or black-box numbers.
